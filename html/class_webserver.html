Detailed Description

Webserver class refers to everything involving requests. This serves as a object that includes wName, start_time, current_time, currentRequest, and *q.

Checks the status of a specific webserver and request. Makes sure the current request in the specific webserver is completed before exchanging the request for a new one. If the request is completed on the specific webserver, it will output a statement to the output file then replace the request with the new one
