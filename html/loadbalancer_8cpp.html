The main load balancer fucntion that performs the load balancer duties. This function will call the other classes, Webserver and Request, to simulate the load balancer. The user will be prompted to enter how many webservers they wish to create and how many clock cycles they would like the load balancer to cycle through. The output for the load balancer will be written into a newly created file named, "LoadBalancerLog.txt"
